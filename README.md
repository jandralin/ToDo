# ToDo App

## Описание

ToDo App — это простое приложение для управления задачами, разработанное с использованием TypeScript и React. Приложение позволяет добавлять, отмечать и удалять задачи, а также хранить их состояние.

## Технологии

- React
- TypeScript
- Webpack
- CSS
